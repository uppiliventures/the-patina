# THE PATINA

999 procedural metal surfaces that live entirely on-chain.
They oxidise continuously. A polish arrests decay for 30 days.
Nothing reverses. The image is not a file. It is a view function
with a clock inside it. Kill every website on earth and The Patina
still renders, still oxidising, from the contract.

Read the [whitepaper](WHITEPAPER.md).

make the rust run backwards, we want to know. Every finding gets

with your name on it.

at day 90.
